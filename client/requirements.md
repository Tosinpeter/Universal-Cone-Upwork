## Packages
react-speech-recognition | Easy React hook for browser SpeechRecognition API
regenerator-runtime | Required dependency for react-speech-recognition
framer-motion | For smooth UI transitions and microphone pulsing effects
recharts | For visualizing score breakdowns and performance metrics
clsx | Utility for conditional class names
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
The app relies on the browser's native SpeechRecognition API.
This works best in Chrome/Edge/Safari.
TTS (Text-to-Speech) will use the browser's window.speechSynthesis API.
